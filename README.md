# Reku - Tarjetas de Lealtad Digitales

Nueva landing page premium para **Reku**, un servicio SaaS que permite a los negocios físicos y locales fidelizar a sus clientes mediante tarjetas de lealtad digitales integradas directamente en **Apple Wallet** y **Google Pay**.

Este sitio web ha sido diseñado con una estética moderna e inmersiva, optimizada para conversiones y velocidad de carga.

## 🚀 Características Clave

* **Simulador de Tarjeta en Tiempo Real:** Los comercios pueden experimentar cómo luciría su propia tarjeta de sellos o cashback directamente en la interfaz móvil.
* **Diseño Premium:** Uso de glassmorphism, gradientes fluidos y efectos de inclinación 3D en dispositivos.
* **Optimización SEO:** Estructura semántica, metatags de redes sociales y descripciones únicas.
* **Acceso de Login Unificado:** Enlace optimizado al portal de acceso centralizado de la plataforma de marca blanca para evitar conflictos de roles.
* **Totalmente Responsivo:** Diseñado desde cero para una experiencia excelente tanto en dispositivos móviles como de escritorio.

## 🛠️ Tecnologías Utilizadas

* **HTML5** semántico y estructurado.
* **Vanilla CSS** con variables personalizadas y animaciones optimizadas.
* **JavaScript (ES6+)** interactivo y eficiente sin dependencias externas.
